# 視窗（二）簡易電子琴

### 專案簡介:
> 模擬真實鋼琴音階的互動式程式，透過呼叫系統底層核心 API 實現發聲功能。程式介面具備動態縮放機制，能根據視窗大小自動調整琴鍵比例。同時，為了優化使用者經驗，程式在關閉前設有防呆二次確認對話框，確保操作的嚴謹性。

### 執行說明:
> 1. 彈奏音階：點擊介面上的琴鍵（Do、Re、Mi、Fa、So、La、Si、Do），程式會根據預設頻率陣列發出對應音效。
> 2. 視窗縮放：拖動視窗邊緣改變大小時，內部琴鍵會透過預存的 Rect 原始位置資料進行等比例縮放，維持版面整齊。
> 3. 結束程式：關閉視窗時會彈出「防呆確認視窗」，使用者點選「是」後才會正式終止程式運行，有效避免因誤觸導致的意外關閉。
